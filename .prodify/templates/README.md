# Prodify Artifact Templates

These templates define the canonical markdown shapes for the task artifacts used by the Prodify workflow.

## Templates
- `orientation_map.template.md`
- `diagnostic_report.template.md`
- `architecture_spec.template.md`
- `refactor_plan.template.md`
- `implementation_summary.template.md`
- `validation_report.template.md`

## Usage
Each task should write its output using the matching markdown template shape from this folder.

## Notes
- Keep headings stable.
- Prefer filling explicit sections over freeform prose.
- When data is unknown, leave conservative placeholders or note uncertainty explicitly.
